# Delta Spec: Login Subtitle

## ADDED Requirements

### Requirement: Subtitle Display

The Login Screen MUST display an engaging subtitle beneath the main title.

#### Scenario: Subtitle Visible

- **Given** the Login Screen is active
- **Then** a subtitle label MUST be visible below the title
- **And** it MUST be centered horizontally
- **And** it MUST use ~18px font size
- **And** it MUST use a muted grey color.
